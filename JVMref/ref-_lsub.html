lsub

subtract two longs

Jasmin Syntax 


    lsub
Stack

Before

After
value1-word1
result-word1
value1-word2
result-word2
value2-word1
...
value2-word2

...

Description

Pops two long integers from the stack, subtracts the top one from the second (i.e. computes value2 - value1), and pushes the result back onto the stack.

Bytecode

Type

Description
u1
lsub opcode = 0x65 (101)
