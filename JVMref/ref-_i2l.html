i2l

convert integer to long integer

Jasmin Syntax 


    i2l
Stack

Before

After
int
long-word1
...
long-word2

...
Description

Pops an integer off the operand stack, sign extends it into a long integer, and pushes the two-word long back onto the stack.

Bytecode

Type

Description
u1
i2l opcode = 0x85 (133)
