isub

subtract two integers

Jasmin Syntax 


    isub
Stack

Before

After
value1
result
value2
...
...

Description

Pops two ints off the operand stack, subtracts the top one from the second (i.e. computes value2 - value1), and pushes the int result back onto the stack.

Bytecode

Type

Description
u1
isub opcode = 0x64 (100)
